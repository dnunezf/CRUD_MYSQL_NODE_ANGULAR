Notas:

Para iniciar un proyecto backend en node.js:

EN TERMINAL DE NUESTRA CARPETA BACKEND:

npm init

Dependencias instaladas:

npm install express mysql2 nodemon cors

Para iniciar servidor express:

En package.json

  "scripts": {
    "dev": "nodemon server.js",
  }

Desplegar servidor express

npm run dev

Para crear proyecto en angular:

ng new -project_name-

ng serve

Creacion de clases en Angular:

ng generate class models/usuario (models es package y usuario es la clase)

Creacion de servicios en Angular:

ng generate service services/usuario

Para iniciar repositorio en GitHub:

En terminal raiz:

git init
git branch -M main

Solucionar para crear un mono-repo:

# 1) Quita el .git interno de Angular
cd crud-angular
rm -rf .git
cd ..

# 2) Saca del índice el “gitlink” previo y agrega los archivos reales
git rm --cached -r crud-angular
git add crud-angular

# 3) Commit y push
git commit -m "fix: track crud-angular as normal folder"
git push origin main

Solucionar para crear dos repos:

# dentro de crud-angular
cd crud-angular
git remote add origin https://github.com/tu_usuario/tu_repo_angular.git
git push -u origin main   # o la rama que uses

git submodule add https://github.com/tu_usuario/tu_repo_angular.git crud-angular
git commit -m "feat: add angular as submodule"
git push
